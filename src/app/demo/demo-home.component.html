Mangol is an open source library for combining Angular, Angular Material and OpenLayers to create a modern, responsive interactive GUI for web maps (M stands for Material, ang for Angular and ol for OpenLayers).
In the left panel you can see all the available modules with working examples and sample code snippets. The complete API documentation is available here (generated with Compodoc).
